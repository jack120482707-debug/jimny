# 油品規格

## 引擎

| 項目 | 建議規格 | 黏度 | 容量 | 更換週期 |
|---|---|---|---|---|
| 引擎機油 | API SP / ILSAC GF-6 或同等級 | 5W-30 | 約 3.6 L，含濾芯 | 10,000 km 或 12 個月 |
| 機油芯 | Suzuki OEM 或同規格 | - | 1 顆 | 每次更換機油同步更換 |

## 傳動系統

| 項目 | 建議規格 | 黏度 | 容量 | 更換週期 |
|---|---|---|---|---|
| 手排變速箱油 | API GL-4 | 75W-90 | 約 1.2 L | 40,000 km 或 48 個月 |
| 分動箱油 | API GL-4 | 75W | 約 1.0 L | 40,000 km 或 48 個月 |
| 前差速器油 | API GL-5 | 80W-90 | 約 1.35 L | 40,000 km 或 48 個月 |
| 後差速器油 | API GL-5 | 80W-90 | 約 1.5 L | 40,000 km 或 48 個月 |

## 煞車與冷卻

| 項目 | 建議規格 | 容量 | 更換週期 | 備註 |
|---|---|---:|---|---|
| 煞車油 | DOT 4 | 約 0.7 L | 40,000 km 或 24 個月 | 開封後容易吸濕 |
| 冷卻液 | Suzuki Super Long Life Coolant 或同等品 | 約 5.4 L | 80,000 km 或 48 個月 | 避免混用不相容冷卻液 |

## 庫存

| 名稱 | 品牌 | 數量 | 是否開封 | 位置 | 備註 |
|---|---|---:|---|---|---|
| 5W-30 全合成機油 | 範例品牌 | 4 L | 否 | 車庫層架 | 下次保養可用 |
| DOT 4 煞車油 | 範例品牌 | 1 L | 是 | 工具櫃 | 已開封，建議優先使用或汰換 |

## 注意

以上資料為範例，實際規格與容量請以車主手冊或維修手冊為準。更換油品後應確認洩漏、液面與試車狀態。
